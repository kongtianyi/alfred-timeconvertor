# alfred-timestamp
A alfred workflow to deal with timestamp convert, including i18n features.

Workflow file are in the repo's root derictory, named `Timestamp I18N.alfredworkflow`.

## Input Format:
```
ts [timezone: utf8] (now|timestamp|datetime)
```
